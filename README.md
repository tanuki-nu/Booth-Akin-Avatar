# akin-avatar
An Akinator-style recommendation system to find your perfect VRChat avatar from BOOTH.
いくつかの簡単な質問に答えるだけで、BOOTHに出品されている膨大な3Dモデルの中から、あなたの好みや理想に最も近い（=Akinな）VRChat向けアバターを導き出すアキネーター形式のレコメンドシステムです。

---

## 🌟 概要 (Overview)
「アバターが多すぎて、自分にどれが合うのかわからない」
「まだ見ぬ理想の3Dモデルに出会いたい」

そんなVRChatユーザーのために、いくつかの選択式の質問に答えていくだけで、システムが条件を自動で絞り込み、最適なBOOTHのアバターを提案します。

---

## ✨ 主な特徴 (Features)
* **アキネーター風の質問UI**: 直感的に選択肢を選んでいくだけで、推論エンジンがおすすめを絞り込みます。
* **「Akin（類似・親近感）」な発見**: 自分が普段探さないような、でも好みにドストライクなアバターとの予期せぬ出会いを提供します。

---

## 📊 収録データについて (Data Specifications)
本システムが検索・推薦に使用しているデータは、以下の仕様に基づいています。

* **データ収集範囲**
  BOOTHにて「3Dキャラクター」で検索した際の、**1ページ目から20ページ目**にヒットするデータを対象にしています。
* **データの更新頻度**
  新着アバターや人気の変動に対応するため、データは**毎月定期的に更新**されます。
* **ご利用時の注意点**
  BOOTHの検索仕様上、収集データ内に一部アバター以外のアイテム（衣装、小物、周辺ツールなど）が稀に混入してしまう場合があります。あらかじめご了承ください。
